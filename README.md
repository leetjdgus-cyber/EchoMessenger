# (C# 코딩) 에코 메신저

## 개요
- C# 프로그래밍 학습
- 핵심기능 : 사용자의 키보드 입력을 받아 입력을 처리하는 프로그램을 구성한다.
- 화면구성 : listbox, textbox, button, label

-과제 내용
- label, textbox, button, listbox를 배치하고, 전송 버튼을 클릭 시 textbox의 텍스트를 listbox로 추가한다.
- 추가 후에 textbox에 남은 텍스트를 비운다.

-구현 내용과 기능 설명
- 입력창에 메시지를 입력하면, 위의 리스트박스에 표시가 되고, 반복하면 메시지가 쌓인다.
